# Student Management System (C)

A simple console-based program written in C for managing student information.  
This project allows users to add, display, search, edit, delete students, and export the data to a file.

---

## Features

- Add new students  
- Display all students  
- Search students by ID  
- Edit student information (ID, Name, Age)  
- Delete student by ID  
- Export student list to a text file

---

## How to Run

Compile the code using GCC:
```bash
gcc main.c -o student_manager
